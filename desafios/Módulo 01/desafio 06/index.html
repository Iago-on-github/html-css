Descrição das Tags acima. Para que servem?

Agora que já conhecemos essa TAGS, veremos a utilidade das mesmas.

  • Parágrafo: A tag <p> serve para criamos um parágrafo.
  • Títulos: A tag <h1> serve para criamos títulos no documento.
  • Imagens: A tag <img scr> serve para adcionarmos imagens no documento.
  • Abreviações: A tag <abbr> serve para adicionarmos abreviação no documento.
  • Código-fonte: A tag <code> serve para colocarmos letras do mesmo tamanho no documento. Deixando, assim, mais fácil e confortável a leitura sobre certos assuntos.
  • Listas numeradas: A tag <ol> serve para adicionarmos uma Lista Numerada, onde, a ordem dos elementos citados importa.
  • Listas com demarcações: A tag <ul> serve para adicionarmos uma Lista não ordenada. Nessa lista, não importa a ordem dos elementos.
  • Citações: A tag <q> serve para adicionarmos ASPAS à uma citação, como por exemplo, a de um livro.
